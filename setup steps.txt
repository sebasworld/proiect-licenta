how to run postgresql in docker:
1. install docker desktop
2. make docker hub account
3. run in terminal/cmd: docker login
4. run : docker run --name postgres-image -p 5432:5432 -e POSTGRES_PASSWORD=postgres -d postgres
5. verify with: docker ps
date psql - user:postgres, db:postgres, pw:postgres, nume con in vs code: PSQL Conn Docker

how to add psql in vs code:
1. install Postgresql Management tool extension
2. ctrl shift p - new query
3. add connection de jos stanga si introdus datele pt postgres din docker si ma conectez la db
4. testare cu : select version()
